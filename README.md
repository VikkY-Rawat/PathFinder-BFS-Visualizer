FS Pathfinding Visualizer

A Python-based interactive application that demonstrates the working of the Breadth-First Search (BFS) algorithm using Pygame. This project helps users understand how BFS explores nodes level-by-level to find the shortest path in a grid.

🚀 Features
📍 Set Start Node and End Node
🧱 Create Walls/Obstacles
🔍 Visualize BFS algorithm step-by-step
🎨 Color-coded nodes for better understanding
🖱️ Interactive UI using mouse controls
⚡ Real-time shortest path generation
🛠️ Technologies Used
Python
Pygame
Data Structures (Queue, Graph Traversal)
📌 How It Works
The grid represents a graph where each cell is a node.
BFS starts from the source node and explores all neighboring nodes level by level.
It guarantees the shortest path in an unweighted grid.
The algorithm uses a queue to keep track of nodes to visit next.
Once the destination is reached, the shortest path is traced back and displayed.
🎮 Controls
Left Click → Set Start / End / Walls
Right Click → Remove Node
Spacebar → Start Visualization
C Key → Clear Grid
